1. W jaki sposób można określić właściwość danego obiektu (dwa sposoby wraz z przykładami)?

Sposób 1 – zapis z kropką:

auto.marka = "Opel";

auto.rok = 2005;

Sposób 2 – zapis w nawiasach:

auto["cena"] = 25000;

2. W jaki sposób można odwołać się do pól (właściwości) i metod?

console.log(samochod.kolor);

samochod.start();

3. Wymień cechy charakterystyczne konstruktora.

- Wywołuje się automatycznie przy tworzeniu obiektu

- Ma taką samą nazwę jak funkcja tworząca obiekt

- Nie zwraca wartości

- Może przyjmować argumenty

- Tworzy obiekt przy użyciu słowa new

4. Konstruowanie obiektu na podstawie przykładu – obiekt ma trzy pola oraz jedną metodę.

var osoba = {
nazwisko: "Kowalski",
imie: "Jan",
zawod: "piekarz",
wyswietl: function() {
document.write(this.nazwisko + " " + this.imie);
}
}

5. Znaczenie słowa this. Jak oddzielamy pola oraz metody?

Słowo this odnosi się do aktualnego obiektu i pozwala korzystać z jego pól i metod.

Pola i metody oddzielamy przecinkami.

6. Konstruktor w JS na podstawie przykładu wraz z opisem.

function Klient(nazwisko, imie, zawod) {
this.nazwisko = nazwisko;
this.imie = imie;
this.zawod = zawod;
this.wypisz = function() {
alert(this.nazwisko + " " + this.imie);
}
}

Opis: Konstruktor tworzy obiekt i przypisuje wartości do jego właściwości.

7. Znaczenie słowa kluczowego new wraz z przykładem.

Słowo new tworzy nowy obiekt na podstawie konstruktora.

var osoba1 = new Klient("Kowalski", "Jan", "kierowca");
var osoba2 = new Klient("Nowak", "Anna", "sekretarka");

8. Znaczenie słowa kluczowego prototype wraz z przykładem i opisem.

Prototype pozwala dodawać nowe właściwości i metody do wszystkich obiektów utworzonych z danego konstruktora.

function Klient() {
this.nazwisko = "Bielski";
this.imie = "Pawel";
}

Klient.prototype.zawod = "kierowca";
Klient.prototype.pisz = function() {
document.write(this.nazwisko + " " + this.imie);
}

Opis: Metody i właściwości dodane przez prototype będą dostępne we wszystkich nowych obiektach tej klasy.
